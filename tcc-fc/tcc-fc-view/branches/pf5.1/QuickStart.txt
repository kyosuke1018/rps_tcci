# 建立新的 web app
Netbeans
New project...
	Java Web > Web Application
	Project Name: tccidemo
	Enable Contexts and Dependency Injection
	Context Path: /tccidemo
	check-on JavaServer Faces, Components: Primefaces
	Finish

tcc-fc-view.51\base.theme1\* -> tccidemo\

加上所有 tccidemo\lib\*.jar (請注意 primefaces jar 版本是否需更新)
lib\cas-client-core-3.2.0.jar Package不要打勾(不加入war中)

asadmin --host localhost multimode --file resources-local.conf
